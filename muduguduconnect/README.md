# MuduguduConnect 🌿
### Hire locally. Get it done fast.

> Rwanda's platform for connecting employers with trusted local domestic workers.

---

## Project Structure

```
muduguduconnect/
├── src/
│   ├── config/         ← db.js (Sequelize config)
│   ├── controller/     ← auth, worker, review, report, chatbot
│   ├── database/
│   │   ├── models/     ← User, Worker, Review, Report
│   │   ├── migrations/ ← (generated by sequelize-cli)
│   │   └── seeds/      ← demo_data.js
│   ├── docs/           ← API.md
│   ├── middleware/     ← auth.js, upload.js, errorHandler.js
│   ├── routes/         ← auth, worker, review, report, chatbot
│   ├── services/       ← email.service.js
│   └── index.js        ← Express app entry point
├── public/
│   └── index.html      ← Full frontend (HTML/CSS/JS)
├── .env.example
└── package.json
```

## Setup

```bash
# 1. Install dependencies
npm install

# 2. Configure environment
cp .env.example .env
# Edit .env with your MySQL credentials and email config

# 3. Create MySQL database
mysql -u root -p
CREATE DATABASE muduguduconnect;

# 4. Run migrations (auto via sync) or with CLI
npm run migrate

# 5. Seed demo data
npm run seed

# 6. Start development server
npm run dev

# 7. Open browser
http://localhost:3000
```

## Tech Stack
- **Runtime**: Node.js
- **Framework**: Express.js
- **ORM**: Sequelize (MySQL2)
- **Auth**: JWT + Bcrypt
- **Email**: Nodemailer
- **Upload**: Multer
- **Frontend**: HTML / CSS / Vanilla JS (mobile-first)

## Key Features
- Worker search by job type, district, availability
- Smart chatbot (EN/RW/FR keyword extraction)
- Emergency mode (online workers only)
- WhatsApp direct contact
- Email verification with 6-digit code
- Rating & review system
- Trust score breakdown
- Report system
- Boost profile UI (Premium)
- Multi-language toggle: EN / RW / FR
- Mobile-first responsive design
- Green color palette from logo
